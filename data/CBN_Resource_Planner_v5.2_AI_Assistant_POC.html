Template CSV format (Projects, Capacity, Cost, and Monthly Capacity)

The Upload Priority CSV can update (1) project order, (2) target capacity per country, (3) country cost values, and (4) monthly capacity overrides per country/month. You can mix these in a single file.

Supported columns (case-insensitive):
• Project: Project, Project Name
• Priority: Priority, Rank, Order
• Country: Country
• Target Capacity: Target Capacity, Capacity, HC Target
• Country Cost: Country Cost, Cost, Cost Multiplier, Cost per HC
• Monthly Capacity: use Month (e.g., Jan 26, 2026-01, 01-Jan-2026) and Monthly Capacity (or MonthlyCap, Monthly HC).

How updates are applied:
• Rows with a Project value set the project order (lowest Priority = highest).
• Rows with a Country can update flat Target Capacity and/or Country Cost (Project can be blank).
• Rows with Country, Month, and Monthly Capacity set per-month capacity overrides. This automatically enables Monthly Capacity mode. Unknown countries are added; months are normalized (e.g., "2026-01" → "Jan 26").
• Blank cells are ignored. Unknown projects are skipped.

Tip: Click Download Priority Template to get a CSV prefilled with your current project list, country capacities, costs, and a section for monthly capacity by country/month.
